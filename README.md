# 图书馆管理系统 python336

## 项目概述

图书馆管理系统是一款基于Python GUI的软件，用于管理图书馆的日常运作，包括用户管理、图书信息存储、采购、淘汰和租借等功能。

## 技术栈

- Python
- PyQt5（图形用户界面）
- MySQL（数据库）

## 功能模块

### 用户管理

- 用户注册
- 用户登录
- 修改密码

### 图书管理

- 存储图书信息
- 采购和淘汰情况管理
- 租借情况管理

### 功能实现

- 图书采购
- 图书淘汰
- 图书租借
- 图书信息查询
- 采购、库存、淘汰、租借情况查询
- 图书相关统计

## 系统角色

- 系统管理员：负责系统的维护与用户管理。
- 图书馆工作人员：负责图书的采购、淘汰、租借等日常操作。

## 运行环境
